# Marketing Funnel Analysis Dashboard

## Project Overview
This project analyzes a marketing campaign dataset to understand customer conversion patterns and identify factors that influence campaign success.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Dashboard Design

## Dashboard Insights
- Majority of customers did not subscribe to the campaign.
- Cellular contact generated the highest conversions.
- Certain months achieved better campaign performance.
- Secondary education customers formed the largest customer segment.
- Conversion rates varied across customer groups.

## Recommendations
- Focus marketing efforts on high-performing contact channels.
- Increase campaigns during high-conversion months.
- Target customer segments with better conversion potential.
- Personalize offers based on customer demographics.
- Improve engagement strategies for low-converting groups.
